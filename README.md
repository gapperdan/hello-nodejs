# hello-nodejs
Some sample node js apps

### Basic example
```
node helloworld.js
```

## Sample code to call a rest client
request_sample.js

### Requires request package
```
npm install request
```

### To run
```
node request-get.js
```

### Sample code to start a node server
```
node server.js
```

### Then hit the url:
```
http://localhost:8888
```
